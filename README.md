# Pemrograman 1 - Teknik Informatika

Repositori ini berisi kumpulan tugas dan latihan praktikum mata kuliah **Pemrograman 1** pada semester 4 (Tahun Kedua) di Universitas Pamulang.

## 📂 Struktur Repositori
Setiap file dalam repositori ini mewakili modul atau pertemuan tertentu selama masa perkuliahan:

* **SourcePackage/**: Berisi file kode sumber (`.java`) untuk setiap latihan dan tugas.
    * `P1_...`: Latihan Pertemuan 1.
    * `P2_...`: Latihan Pertemuan 2.
    * `P3_...`: Latihan Pertemuan 3.
    * `P4_...`: Latihan Pertemuan 4 & Tugas Mandiri.
    * `P5_...`: Latihan Pertemuan 5.
    * `P6_...`: Latihan Pertemuan 6.

## 🚀 Teknologi yang Digunakan
* **Bahasa Pemrograman**: Java
* **Version Control**: Git & GitHub
* **Environment**: Windows Command Prompt (CMD)
